Group Project

CODE REVIEW TRAINING APP FOR C#

Project Overview

The project aims to develop a web-based code review training application for the Software Engineering module (SET09102) at Edinburgh Napier University. The application will provide students with interactive exercises to practice identifying and resolving code defects. The system will present code snippets with known issues, allow students to analyze and propose improvements, and provide immediate feedback. Advanced features, such as compiling and running refactored code, are considered stretch goals.

Objectives

  • Enhance Learning Experience: Create an interactive platform for students to practice code review.
  • Immediate Feedback: Provide real-time feedback on students' analysis and proposed solutions.
  • Code Refactoring: Allow students to propose and receive feedback on refactored code.
  • User Accounts: Enable students to register and save their progress.
  • Graded Problems: Implement a system to grade problems by difficulty and create levels for progression.
  • Scalability: Design the system to easily add new problems over time.

Deliverables

  • Working Prototype: A functional web-based application for code review training.
  • Source Code: Complete source code stored in an appropriate repository.
  • Graded Problems: A set of code review problems categorized by difficulty.
  • Documentation: Comprehensive documentation covering system setup, usage, and maintenance.

Methodologies

  • Web Development: Utilize web design and development techniques to create the application.
  • Project Management: Implement project management practices to ensure timely delivery.
  • System Configuration: Configure the system on a Linux server provided by the university.
  • Interactive Design: Focus on user interactivity and immediate feedback mechanisms.
  • Code Analysis: Develop algorithms to analyze and provide feedback on code refactoring.
  • User Authentication: Implement user registration and login functionality to save progress.
  • Scalable Architecture: Design the system to allow easy addition of new problems and features.

Key Features

C# Reviewer

The application would process, review, and output feedback regarding C# code given to it.

C# Educator

Interactive quizzes would offer the user opportunities to test their existing C# knowledge.

Sleek Design

The design took inspiration Napier Universities iconic colour palette - ensuring familiarity and professonalism.

User Personalisation

The ability to create unique user accounts to store and view results - additional features included dark/light mode for user preference.
